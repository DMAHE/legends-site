# LEGENDS IT — Company Website

A neat, modern, responsive static website for **LEGENDS IT**, an AI and Tech Services company. Built with semantic HTML5, custom CSS3 (Flexbox/Grid, CSS variables, animations), and vanilla JavaScript — no frameworks, no build step.

## Run locally

Just open `index.html` in any modern browser. Optionally serve it:

```bash
# Python
python -m http.server 8000
# then visit http://localhost:8000
```

## Files

| File | Purpose |
| --- | --- |
| `index.html` | Markup and content |
| `style.css` | Theme tokens, layout, components, responsive rules |
| `script.js` | Sticky header, mobile menu, reveal-on-scroll, carousel, particles |

## Sections

Header/nav, Hero, Client marquee, Services, Industries, Success stories carousel, Insights, Footer.
